## 체크리스트 정리
- 구현할 기능

  회원(회원 가입 및 수정, 회원 조회), 상품 (상품 등록 및 수정, 재고 관리), 주문 (주문 생성, 주문 취소), 결제 (결제 요청, 결제) 기능을 구현한다.

  너무 많은 기능을 구현하기보다 예외 처리나 성능(쿼리 N+1 문제 등)이 중요하기 때문에 기능을 적당히 구현하고, 추가 기능이 필요하다는 생각이 들면 할인 (선착순 쿠폰) 기능을 추가해 볼 생각이다.


- 구현하지 않을 기능

  리뷰, 추천 기능을 만들지 않는다.

  위의 설명과 비슷한 이유로 두 기능은 본 프로젝트의 핵심 도메인과는 연관이 적은 기능들이어서 구현하지 않는다.


- MVP 기준

  회원이 상품을 주문하고, 재고가 차감되고, 결제가 처리된다.


- Git 저장소 생성, GitHub 원격 저장소 연결, 첫 커밋 완료
